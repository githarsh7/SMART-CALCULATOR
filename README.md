# Responsive Keyboard Calculator

A modern and responsive calculator built using HTML, CSS, and JavaScript.  
This project focuses on keyboard event handling, expression evaluation, input validation, and responsive UI design.

---

# Features

- Keyboard-controlled input
- Button click support
- Full inline expression evaluation
- Separate functions for all operations
- Invalid key blocking with alert messages
- Divide-by-zero protection
- Responsive design for desktop, tablet, and mobile
- Modern UI using Flexbox and media queries

---

# Supported Operations

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Modulus (`%`)

---

# Keyboard Controls

| Key | Action |
|------|--------|
| `0-9` | Number input |
| `+ - * / %` | Operators |
| `Enter` | Calculate result |
| `Backspace` | Delete last character |

Invalid key presses are blocked and display a warning alert.
